RStudio Desktopのインストール

RStudio Desktopはローカル環境で R を効率よく実行するための統合開発環境（IDE）です。 R を使いこなすための様々な機能が実装されていますので、 R を使う場合にはおすゝめの環境です。

インストール

RStudio Desktopのインストール自体は対話的に進めれば問題なくインストールできます。 こちら からプラットフォームに見合ったファイルをダウンロードしてください。 　

エラーが出る場合

環境によってはLinuxのライブラリが不足してインストールエラーになる場合があります。その場合は、エラーメッセージで表示されたライブラリをインストールしてから再度RStudioをインストールしてください。
