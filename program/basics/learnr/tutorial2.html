Introduction

本ファイルはデータ分析勉強会で使用した『RStudioのすゝめ』の講義資料を再整理しlearnrパッケージを用いて編集しなおした初学者向けのチュートリアルです。実際にRのコードを入力して実行することが可能ですがshinyサーバが必要となりますのでRStudioにて実行してください。

実行方法

本ファイルをRStudioで開くと上段に[Run Document]というボタンが表示されますので、これをクリックしてください。コンパイル後にローカルホスト(127.0.0.1)上にshinyサーバが起動し[Viewer]ウィンドウに実行可能なファイルが表示されます。ファイルが表示されたら[Code]と左上に書いてある欄に実行コードを記述し右側の[Run Code]ボタンを押せば記述されている（記述した) コードが実行されます。当然、間違っている場合はエラーがでます。コードを追加・記入したい場合は最終行の行末にカーソルを持っていき改行することで追加ができます。 学習結果は記録されますがリセットしたい場合はTOCの一番下にある[Start Over] をクリックしてください。

なお、learnrパッケージの多国語対応が不十分なためメニューを英語表記にしていますが各Chapterは以下のようになっています。

Chapter 6 - nclass, pretty, cut, table

度数

度数はデータの分布を確認するために使う度数分布表やヒストグラムを作る上で必要となる統計量の一つです。度数を求めるためには度数を求めるための区間である階級（幅）を定める必要があります。
階級はデータに合わせて任意に決めることができますが、なんらかの基準を用いて階級を決める場合にはスタージェスの公式やスコットの選択、フリードマン＝ダイアコニスの選択を用いて階級数を求めてから階級を求めます。

階級数を求める

スタージェスの公式（nclass.Sturges関数）を用いてirisデータセットの四つの数値データに対する階級数を求めて見ましょう。

nclass.Sturges(x)
nclass.scott(x)
nclass.FD(x)
sapply(iris[, -5], FUN = nclass.Sturges)

階級(幅)を求める

階級数が求められましたので、これを元に階級(幅)を求めてみましょう。度数分布表やヒストグラムを作るのが最終目標ですから単純にデータの範囲を階級数で分割してしまうと都合が悪くなりますので、ここでは階級境界値(Breakpoints)を計算するpretty関数を用いてirisデータセットの四つの数値データに対する階級境界値を求めて見ましょう。

pretty(x, n = 5, min.n = n %/% 3,  shrink.sml = 0.75,
       high.u.bias = 1.5, u5.bias = .5 + 1.5*high.u.bias,
       eps.correct = 0, ...)
pretty(range(iris$Sepal.Length), n = nclass.Sturges(iris$Sepal.Length))
pretty(range(iris$Sepal.Width), n = nclass.Sturges(iris$Sepal.Width))
pretty(range(iris$Petal.Length), n = nclass.Sturges(iris$Petal.Length))
pretty(range(iris$Petal.Width), n = nclass.Sturges(iris$Petal.Width))

階級を知る

度数を求める前にirisデータセットの四つの個々の数値データがどの階級に含まれるのかを知る必要がありますが、この処理にはcut関数を用います。

cut(x, breaks, labels = NULL,
    include.lowest = FALSE, right = TRUE, dig.lab = 3,
    ordered_result = FALSE, ...)
cut(iris$Sepal.Length, breaks = pretty(range(iris$Sepal.Length),
    n = nclass.Sturges(iris$Sepal.Length)))
cut(iris$Sepal.Length, breaks = pretty(range(iris$Sepal.Length),
    n = nclass.Sturges(iris$Sepal.Length)))
cut(iris$Sepal.Length, breaks = pretty(range(iris$Sepal.Length),
    n = nclass.Sturges(iris$Sepal.Length)))
cut(iris$Sepal.Length, breaks = pretty(range(iris$Sepal.Length),
    n = nclass.Sturges(iris$Sepal.Length)))

度数集計

これによりirisデータセットの四つの個々の数値データに対する階級が求められましたので、このデータを元にtable関数を用いて度数(分布表)を求めます。

table(...,
      exclude = if (useNA == "no") c(NA, NaN),
      useNA = c("no", "ifany", "always"),
      dnn = list.names(...), deparse.level = 1)
table(cut(iris$Sepal.Length,

                          n = nclass.Sturges(iris$Sepal.Length))))

          breaks = pretty(range(iris$Sepal.Length),
                          n = nclass.Sturges(iris$Sepal.Length))))
table(cut(iris$Sepal.Length,
          breaks = pretty(range(iris$Sepal.Length),
                          n = nclass.Sturges(iris$Sepal.Length))))

検定 - 分散分析

三群以上で平均値の差を検定する分散分析を行うにはoneway.test関数を用いるのが便利です。

oneway.test(formula, data, subset, na.action, var.equal = FALSE)
irisデータセットを用いて、種別（Species）毎の花弁（Petal）の平均値の差を検定してみましょう。 
oneway.test(Petal.Length ~ Species, data = iris)
boxplot(Petal.Length ~ Species, data = iris)    # 分布を可視化して確認
oneway.test(Petal.Width ~ Species, data = iris)
boxplot(Petal.Width ~ Species, data = iris)     # 分布を可視化して確認

Chapter 18 - prop.test

検定 - 比率

比率の検定には一標本の比率の検定と二標本の比率の検定があります。一標本の場合は比率の指定を省略した場合はp = 0.5すなわち標本内の層別データの比率に対する検定となります。一方、二標本の場合は標本間の比率に対する検定となります。

prop.test(x, n, p = NULL,
          alternative = c("two.sided", "less", "greater"),
          conf.level = 0.95, correct = TRUE)

一標本の比率の検定

titanicデータセットを用いて、生存者（Survived）に対する1標本の比率の検定を行ってみましょう。Titanicデータセットはirisデータセットやmtcarsデータセットと異なり、table形式なのでベクトル形式への変換結果を利用して下さい。 
x <- rbind(xtabs(Freq ~ Survived, data = as.data.frame(Titanic)))
x <- rbind(xtabs(Freq ~ Survived, data = as.data.frame(Titanic)))
prop.test(x)
barplot(x)      # 分布を可視化して確認

二標本の比率の検定

titanicデータセットを用いて、生存者（Survived）と年齢（age）に対する2標本の比率の検定を行ってみましょう。Titanicデータセットはirisデータセットやmtcarsデータセットと異なり、table形式なのでベクトル形式への変換結果を利用して下さい。 
x <- rbind(xtabs(Freq ~ Survived + Age, data = as.data.frame(Titanic)))
x <- rbind(xtabs(Freq ~ Survived + Age, data = as.data.frame(Titanic)))
prop.test(x)
barplot(x)      # 分布を可視化して確認

Chapter 19 - dnorm

分布 - 正規分布

Rでは様々な密度関数が用意されています。正規分布の場合はdnorm関数です。

dnorm(x, mean = 0, sd = 1, log = FALSE)

また、任意の関数（数式）に対する任意の入力範囲に対する出力を描く関数も用意されています。なおexprに対する引数はxで渡します。その他、詳細はRのヘルプで確認してください。

curve(expr, from = NULL, to = NULL, n = 101, add = FALSE,
      type = "l", xname = "x", xlab = xname, ylab = NULL,
      log = NULL, xlim = NULL, ...)
平均値50、標準偏差10の正規分布の確率密度分布を描いてみましょう。描く範囲は任意としますが、なるべく分布の形が分かるような範囲として下さい。 
curve(, from = , to = ) 
curve(dnorm(x, mean = 50, sd = 10), from = 20, to = 80)
