Program

ProgramはData Science Workflow において全ての分析プロセスで利用する道具・環境を意味しています。


　
データ分析における基本的で共通な処理はプログラミングで処理することにより分析自体を容易にするだけでなく分析による問題解決に集中できる時間を与えてくれると共に分析の再現性を確保します。 なお、分析の再現性（Reproducible Research）を実現するためのツールについてはCommunicationで扱う予定です。
　

分析の基本

データ分析の基本についてはModel/Inferで扱う予定です。
　

分析ツール

分析に利用するツールとしてR 、R Commander ならびにRStudio Desktop を前提としています。分析環境の導入方法やツール、ライブラリ類に関する情報、環境の基礎を提供する予定です。各ページへは最上段のメニューからアクセスしてください。なお、R の個々のパッケージの使い方についてはTidy/TransformやModel/Inferなどの各分析プロセスのページにて扱います。
　

Rとは

R とは統計計算、データ分析、計算・分析結果の可視化に特化したオープンソースの言語であり分析環境です。R は動的言語と言われ、厳密なデータ型の定義を行わなくても処理が可能で、データ・フレーム型と呼ばれるデータ分析に非常に適したデータ構造を扱えます。また、近年では機械学習関連のパッケージが充実しMicrosoftやOracleなどの大手IT企業もR に注力しています。
R の詳細についてはWhat is R? をご覧ください。
　

なぜRなのか

R は他の言語や分析環境に比べると以下のような点が強みがあります。

  1. 多様な統計手法を網羅する多数パッケージが提供されている
  2. 統計手法だけでなく可視化のためのパッケージも多数提供されている
  3. データ分析に適したデータ構造を始めとして柔軟性が高い設計になっている
  4. 分析結果をそのままレポートにできる仕組みがあり分析再現性が高い
  5. 他言語（C/C++, Python, Stanなど）のコードを比較的簡単に呼び出せる
  6. 利用者、開発者による活発なコミュニティが形成されており情報を入手しやすい
  7. R に特化した統合開発環境もオープンソースで提供されている

このような他言語にはない強みを活かしたインタラクティブな分析が可能でアドホック分析に非常に適しています。また、ネットコミュニティの調査 では他の言語ユーザに比べるとR のユーザは親切だと評価されているようです。
