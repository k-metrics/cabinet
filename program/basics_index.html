Rの基本

データ分析勉強会 で分析演習を行う際には主にR Commander(Rcmdr) パッケージを利用してきました。Rcmdr はコードを書かなくても分析ができるので敷居が低く便利ではありますが、Rcmdr にできることは残念ながら限られています。本来R でできる様々な分析を行うにはRcmdr から離れR のスクリプトの世界に足を踏み入れる必要があります。
　

RStudioを使おう

R Commanderではできない分析を行うにはマルチプラットフォーム対応のRStudio Desktop からR を直接使うのが最も便利だと思います。R /RStudio で出来ることは

  • R Comannderで出来ること全て
  • R Comannderに出来ないこと全て

です。当然といえば当然なんですが、Rcmdr では出来ないがR /RStudio で出来ることを具体的に2017年度のデータ分析勉強会 のテキストである統計学基礎 の第1章の中から例に上げてみましょう。

R/RStudioで出来ることの例 利用パッケージ例
度数分布表の作成と表示 dplyr, knitr, DT
ローレンツ曲線の描画とジニ係数の計算 ineq
カラフルで分かりやすいグラフ ggplot2
クロス集計表の作成と表示 dplyr, knitr, DT
時系列分析 TTR

その他、RStudio を使うことでR の機能をフルに活用できるようになり以下のようなこともできるようになります。

R/RStudioで出来ることの例 利用パッケージ例
高度なデータハンドリング tidyverse
インタラクティブなレポート rmarkdown
Webページの作成 rmarkdown
電子ブックの作成 bookdown
データスクレイピング tidyverse
自然言語処理 RMeCab
ベイズ統計 rstan, mcmc
機械学習 randomForest
深層学習（ディープラーニング） h2o

なお、R における各分野の開発についてはCRAN Task View で公開されています。

チュートリアル資料について

チュートリアル資料はデータ分析勉強会 で使用した『RStudioのすゝめ』の講義資料を再整理しlearnrパッケージを用いて編集しなおした初学者向けのチュートリアルです。実際にRのコードを入力して実行することが可能ですが実行環境は以下の条件を満たす必要があります。

Program Version Repository 設定等
R v3.4 or later CRAN
RStudio v1.0.143 or later RStudio
Package Version Repository 備考
DT v0.2 or later CRAN
formatR v1.5 or later CRAN
knitr v1.16 or later CRAN
learnr v0.9 or later CRAN
rmarkdown v1.6 or later CRAN
shiny v1.0.3 or later CRAN
tidyverse v1.1.1 or later CRAN

その他、チュートリアルファイルに応じたパッケージが必要となりますので、詳細はチュートリアルファイルをダウンロードの上で確認して下さい。
